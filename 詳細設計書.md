1. `useReport(reportId)`

レポート全体情報（タイトル・要約・セクション・クラスタ）を取得する hook。

---

📄 実コード（典型例）

import useSWR from 'swr';
import { getReport } from '@/api/report';
import type { Report } from '@/lib/types';

export function useReport(reportId: string) {
  const { data, error, isLoading } = useSWR<Report>(
    reportId ? `/reports/${reportId}` : null,
    () => getReport(reportId)
  );

  return {
    report: data,
    isLoading,
    error,
  };
}


---

🧠 処理内容（内部動作）

① SWR によるキャッシュキー生成

reportId ? `/reports/${reportId}` : null


• reportId が undefined のときはフェッチしない
• キーが同じなら SWR がキャッシュを返す


② API 呼び出し

() => getReport(reportId)


• api/report.ts の getReport() を呼ぶ
• fetcher → Route Handler → repository → DB の流れでデータ取得


③ ローディング状態管理

isLoading


• UI はこの値を見てスケルトン表示などを行う


④ UI に渡すための整形

• 特に整形は行わず、Report 型のまま返す
• UI は props として受け取るだけ


---

📤 返却型（UI が受け取る）

{
  report: Report | undefined;
  isLoading: boolean;
  error: any;
}


---

2. `useClusters(reportId)`

クラスタ一覧を取得し、UI が扱いやすいように整形する hook。

---

📄 実コード（典型例）

import useSWR from 'swr';
import { getClusters } from '@/api/clusters';
import type { Cluster } from '@/lib/types';

export function useClusters(reportId: string) {
  const { data } = useSWR<Cluster[]>(
    reportId ? `/reports/${reportId}/clusters` : null,
    () => getClusters(reportId)
  );

  const clusters = data?.sort((a, b) => b.comment_count - a.comment_count) ?? [];

  return { clusters };
}


---

🧠 処理内容（内部動作）

① SWR によるクラスタ一覧の取得

• /reports/{id}/clusters をキーにキャッシュ
• getClusters() を呼び出す


② データ整形（UI ロジック）

data?.sort((a, b) => b.comment_count - a.comment_count)


UI が扱いやすいように：

• コメント数の多い順にソート
• null の場合は空配列を返す


③ UI に渡す props を返す

return { clusters };


---

📤 返却型

{
  clusters: Cluster[];
}

4. `useChartData(reportId)`

embedding（散布図座標）を取得し、Recharts 用に整形する hook。

---

📄 実コード（典型例）

import useSWR from 'swr';
import { getEmbedding } from '@/api/embedding';
import type { EmbeddingPoint } from '@/lib/types';

export function useChartData(reportId: string) {
  const { data } = useSWR<EmbeddingPoint[]>(
    reportId ? `/reports/${reportId}/embedding` : null,
    () => getEmbedding(reportId)
  );

  const chartData = data?.map(p => ({
    x: p.x,
    y: p.y,
    cluster: p.cluster_id,
  })) ?? [];

  return { chartData };
}


---

🧠 処理内容

① embedding の取得

• /reports/{id}/embedding をキーに SWR がキャッシュ
• getEmbedding() を呼ぶ


② Recharts 用に整形

{ x: p.x, y: p.y, cluster: p.cluster_id }


UI はこの形式でないと描画できないため、
hooks が責務として整形している。

---

📤 返却型

{
  chartData: { x: number; y: number; cluster: string }[];
}

subgraph ADMIN["🛠 Admin（管理画面 / Next.js）"]
